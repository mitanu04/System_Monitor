# Proiect Sisteme de Operare – System Monitor

## Descriere

Acesta este un proiect realizat pentru disciplina **Sisteme de Operare** în anul 2 de studiu.  
Scopul proiectului este de a implementa un monitor de sistem simplu în limbajul C, care colectează și afișează informații esențiale despre procese și resursele sistemului de operare.

---

## Funcționalități

- Afișează lista de procese active și detalii despre acestea
- Monitorizează utilizarea resurselor sistemului (CPU, memorie)
- Parsează și interpretează date din `/proc`
- Modularitate și structură clară a codului
